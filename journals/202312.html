对于这类尚在实验阶段的 Web 特性，你可以在以下几个地方查找相关文档和信息：

  • MDN Web Docs (Mozilla Developer Network): MDN 通常提供最新的 Web API 文档和示例代码。即使是实验性特性，也可能在这里找到一些基础信息。
  • W3C 或 WHATWG 规范: 这些组织负责制定和更新 Web 标准，你可以直接在这些组织的网站上搜索 FileSystemHandle 或 File System Access API 的规范草案。
  • Chromium Codebase: 如果你对源代码感兴趣，可以查看 Chromium 项目的代码仓库。因为 Chrome 是基于 Chromium 项目开发的，所以相关 API 的实现会在这个仓库中。
  • Chrome Platform Status: 这个网站跟踪 Chrome 提供的所有特性的实现状态和计划，对于正在开发中的特性也会有所提及。
  • Chrome DevTools: 在最新版本的 Chrome 浏览器中，通过 DevTools 的“控制台”或者“源代码”面板，你可以实时查看 API 的实现和行为。
  • Chrome Issue Tracker: 如果是 Chrome 特有的实验性 API，相关的讨论和文档可能会在 Chrome 的问题跟踪系统中出现。
